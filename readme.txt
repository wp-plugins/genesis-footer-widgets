== Description ==
This plugin adds 3 footer widgets to Genesis 1.6 and higher. 
For this plugin to work you will need to have the Genesis theme installed and use it - or a child theme of Genesis.

== Installation ==
1. Upload `gfw/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
